Présentation du mini-projet

Dans le cadre du cours PPE1, nous avons créé un miniprojet consistant à créer une page web hébergée sur GitHub (vous êtes dessus) contenant le lien vers un tableau de résultats (voir ci-dessous). Ce tableau a été obtenu grâce à la création d'un programme en bash qui, étant donné un fichier txt contenant des URLs, renvoie son code http ainsi que son encodage. Pour des raisons de lisibilité, nous avons également associé à chaque URL un numéro. Ce mini-projet a pour objectif de nous préparer à la réalisation du "vrai" projet dans lequel nous ferons une étude comparative de l'usage d'un mot à travers différentes langues.
